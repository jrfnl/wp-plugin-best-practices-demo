== Installation ==

1. Extract the .zip file and upload its contents to the whole `demo-quotes-plugin` folder to your `/wp-content/plugins/` directory. Alternately, you can install directly from the Plugin directory within your WordPress Install.
1. Activate the plugin through the "Plugins" menu in WordPress.
1. Add quotes
1. Add the quotes to your website using a custom menu
